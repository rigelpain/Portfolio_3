PRODUCTS 過去の作品

『 頭鍛冶 』

- Grobal Game Jam 2020 Hakodate -

・テーマ
　「 repair 」

・ツール　言語
　Unity / ESP32 / illust

・関連リンク
けすのん / カブ /
　 GGJ Hakodate / たくみ

2020.01.31 ~ 02.02

両手を失った職人による
魂の鍛冶

世界中で同時開催されるゲーム作りのお祭り「GGJ」
「頭鍛冶」はその世界規模のイベントで最も斬新で頭がクラクラするゲームと言われている。

頭鍛冶のルールは簡単。上から落ちてくる「けすのんノーツ」に合わせて頭を上下に振るだけ！
頭に巻いたハチマキに搭載した角速度センサが頭突きを感知しESP32を介してアプリへ送信するというプロセスで、センサの感度から大きくスピーディに頭突かなければ反応しない。 が、これが程よい難易度を演出してくれた。テストを重ねた開発者達も首を若干痛めたらしい。

GGJから提示されたテーマは1つ「 repair 」。ここから折れた刀を再生させる鍛冶を連想したわけだが、頭鍛冶に行き着く前に惜しくも没になったネタがいくつかある。それらはどれも「体を動かす」タイプのものだった。 IoTにロマンを感じたり、なにより体を動かすのは楽しいので遊びと相性が良いのではとよく考える。

たくみの手助けもあったが、カブは今回初めてUnityを触っており完成まで漕ぎつけたことに感動した。けすのんはロボコニストの腕を活かしいつもハードを作ってくれる。私は今回アイデア出し以降できることがなかったのでずっと絵をかいていた。けすのんイラストは自信作で、本人もデスクトップ画面にしてくれている。

担当
ソフト（Unity）：カブ
ハード（ESP32）：けすのん
デザイン（illust）：me
助っ人Unity使い：たくみ
